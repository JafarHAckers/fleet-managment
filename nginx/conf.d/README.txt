# Fleet Management System (Bus Booking System)

This is a fleet management system (bus booking system) built with Laravel. It allows users to book seats on bus trips between different cities in Egypt.

## Features

- Book a seat on a trip between any two stations
- View available seats for a specific route
- Handles partial trip bookings (e.g., Cairo to AlFayyum, Cairo to AlMinya, etc.)
- Proper validation for booking requests
- Unit and feature tests

## Requirements

- PHP 8.1+
- Composer
- MySQL
- Laravel 10.x
